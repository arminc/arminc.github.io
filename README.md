arminc.github.io
================

Jekyll blog for: blog.coralic.nl

Install
=======
You need 
+ Ruby 1.9 or 2
+ bundler (gem install bundler)

Install dependencies of the project
+ bundle install

Develop
=======
jeklly build
jekkly serve
